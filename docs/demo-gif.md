## Troubleshooting

### OBS Not Recording
- Check permissions (screen recording)
- Restart OBS
- Update graphics drivers

### Low Frame Rate
- Reduce resolution to 1280x720
- Lower FPS to 30
- Close other applications

### Large File Size
- Reduce bitrate to 4000 Kbps
- Use H.265 codec (smaller)
- Compress with HandBrake

### Audio Issues
- Check microphone settings
- Adjust audio levels in OBS
- Use noise suppression filter
