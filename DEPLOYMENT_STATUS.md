## Troubleshooting:

### If you see "Frontend not built" message:
1. Check Render build logs for errors
2. Verify Node.js version is 18+ in Render settings
3. Check that `ui/` folder has all files committed

### If build fails:
1. Look for specific error in Render logs
2. Common issues:
   - Missing dependencies in package.json
   - TypeScript errors
   - Build timeout (increase in Render settings)
